If use 'Open in IE' with firefox/chrome or 'Open File Path' with firefox,
BrowserUtility-HostApp and Python Enviroment are required.

BrowserUtility-HostApp:
Open below url and download 'Source code (zip)'
https://github.com/s-ogawa-github/BrowserUtility-HostApp/releases/latest
Unzip "BrowserUtility-HostApp-Ver_*.zip" and run host_app_install.bat.

Python Enviroment:
If you don't have python enviroment, install 'python3.6' from python.org
or download 'python.exe' from below url and move "python.exe" to hostapp/src/"python.exe".
https://github.com/masamitsu-murase/single_binary_stackless_python/releases/tag/v3.6.8-slp-L22
